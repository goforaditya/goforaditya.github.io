Device

Once I have a decent WebApp (Tauri)[https://v2.tauri.app/], (FlutterFlow)[], (Hotwire Native)[https://native.hotwired.dev/] can help me convert it to an App.

I can run a Linux distro in VirtualBox to create an image that runs the App like a Kiosk/default and just allows some basic functionalities outside the app core. Connecting to Wifi, taking notes, System settings etc., Long term goal is for the App itself to become the GUI layer for the Linux operating system without needing to switch to anything else.

For the pyshical device it will have.

  1. A keyboard. Can directly use Raspberry Pi 500 for this.
  2. A screen. I.e. it will be a cyberdeck.
  3. I/O. Option to add a monitor and open USB and GPIO for extensibility.

Optionally will create dedicated version of internet platform Apps using opensource tech. A Browser. An IDE, a dedicated mastodon, A Q&A platform (Apache Answer)[https://github.com/apache/answer], Realtime P2P messaging. Interest Forums etc,. Also thinking of adding multiple modes of connectivity WiFi and LTE for courses. And BlueTooth and Ham Radio for Device to Device comms.
